flashenpost.de by Aleksander Dymshyts

Filter

1. To get filtered data from backend, please use Filter button.
2. To get filtered data from cache, please use Filter (cache) button. (Only visible if cache is not empty).
3. To clear filtered data cache and see all beers, please use Clear button.

Sort

1. To get sorted asc data from backend, please use Asc button.
2. To get sorted asc data from cache, please use Asc (cache) button. (Only visible if cache is not empty).
3. To get sorted desc data from backend, please use Desc button.
4. To get sorted desc data from cache, please use Desc (cache) button. (Only visible if cache is not empty).
5. To clear sorted data cache and see all beers, please use Clear button.
